# blocking_payments
## Кейс по системной аналитике реализующий запросы бизнеса.
## Описание проекта:
Проект содержит спецификацию API и описание базы данных, реализующих логику блокировки, разблокировки и проверки статусов клиентов и их платежей.
### Папка BD
Описание: Информация по базе данных
1. BD_shema.dbml — модель базы данных в формате DBML (Database Markup Language), предназначенная для визуализации и генерации SQL-схем;
2. Description_BD.md — текстовое описание структуры базы данных, включая таблицы, поля и связи между ними;
3. Structure.png — ER-диаграмма (Entity-Relationship Diagram), наглядно отображающая взаимосвязи между таблицами.
### Папка Spicification
1. Blocking_Client_API.yaml — спецификация OpenAPI (v3.0.3), описывающая API для управления блокировками клиентов и их платежей.
Включает описание эндпоинтов, моделей данных (schemas), примеров запросов и ответов.
2. Description.md — сопроводительное описание спецификации: цели, структура, назначение основных сущностей и примеры использования API.
